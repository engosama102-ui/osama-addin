# Osama Addin — PowerPoint Design Tool

Professional design tools for PowerPoint — alignment, shapes, tables, colors, and typography. Works like Illustrator inside PowerPoint.

## Install
1. Download `manifest.xml`
2. Open PowerPoint → Insert → Get Add-ins → My Add-ins → Upload My Add-in
3. Select the manifest.xml file

## Features
- Smart Alignment & Distribution
- Shape Control (corner radius, opacity, fill, border)
- Table Designer with saved templates
- Typography Panel (tracking, leading, baseline)
- Color Palette
- Canvas Grid & Guides
- SVG Insert
