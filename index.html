Installation
3 steps to install
1

Download the manifest file

Click the download button above to get the manifest.xml file. Save it anywhere on your computer.

2

Open PowerPoint

Go to Insert → Get Add-ins → click My Add-ins tab at the top → Upload My Add-in

3

Select the file and start

Choose the manifest.xml file you downloaded. The add-in will appear in your Home tab immediately.
