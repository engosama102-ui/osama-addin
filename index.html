🪟 Windows Installation

1

Download the manifest file

Click below to download the manifest.xml file. Save it anywhere on your computer.

⬇ Download manifest.xml
2

Open PowerPoint → Insert

Click Insert from the top menu bar

3

Get Add-ins → My Add-ins

Click Get Add-ins → click the My Add-ins tab at the top of the window

4

Upload My Add-in

Click Upload My Add-in → select the manifest.xml file you downloaded

✓ The add-in will appear in your Home tab immediately — ready to use!

🍎 Mac Installation

1

Close PowerPoint first

Make sure PowerPoint is completely closed before continuing

2

Open Terminal

Press Cmd + Space → type Terminal → press Enter

3

Run this one command

Copy the command below, paste it in Terminal, and press Enter:

mkdir -p ~/Library/Containers/com.microsoft.Powerpoint/Data/Documents/wef && curl -o ~/Library/Containers/com.microsoft.Powerpoint/Data/Documents/wef/manifest.xml https://engosama102-ui.github.io/osama-addin/manifest.xml
⚠️ This command only needs to be run once — ever. Updates happen automatically after that.
4

Open PowerPoint

Open PowerPoint → you will find Osama Addin in the Add-ins tab in the top menu

✓ Done! Installed permanently — no Terminal needed again.
